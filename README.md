# CongiRH - نظام إدارة الإجازات

تطبيق متكامل لإدارة الإجازات والغيابات في الشركات.

## البنية

- **backend/** - API Spring Boot (Java)
- **frontend/** - تطبيق React

## التثبيت

### Backend (Java)
\\\ash
cd gestionconges/gestionconges
mvn clean install
mvn spring-boot:run
\\\

### Frontend (React)
\\\ash
cd congirh
npm install
npm run dev
\\\

## الميزات

- تسجيل الدخول والمصادقة
- طلب إجازة جديدة
- إدارة الموظفين
- تقارير وإحصائيات
- إدارة الفريق
